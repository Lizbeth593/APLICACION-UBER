# Aplicacion Uber 
La aplicación de Uber esta diseñada para tener un control de todos los pedidos.
<img width="750" height="450" alt="image" src="https://github.com/user-attachments/assets/0caa83a2-dfdf-45b8-907c-ff691bb16d87" />

## Ejecución del programa
1. Descargar el archivo .exe mediante el link. 
   https://github.com/Lizbeth593/APLICACION-UBER/blob/main/uber.exe
2. Ejecutar el archivo previamente descargado.

## Funcionalidad 
El programa se podrá visualizar mediante una ventana de consola la cual te permitirá realizar diferentes funciones como:
 1) Registrar pedidos
 2) Mostrar pedidos
 3) Actualizar pedido
 4) Eliminar pedido
